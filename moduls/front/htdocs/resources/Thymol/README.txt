1.0.0
=====
- Provides support for almost all Thymeleaf 2.0 standard dialect attributes.
- Variable definition using json-like constructs.
- Introduces mappings for URL definition.
- Introduces structures for messages specification.
- Updated examples. 

0.1.3-SNAPSHOT
=====
- Provides limited support th:src and th:href.
- Updated examples. 

0.1.2
=====
- Supports user control of the path to template files.
- Fixes some problems with substituteby.
- Documentation of browser support capabilities.
- Updated examples. 

0.1.1
=====
- Supports (limited) processing capability for expressions in th:include and th:subtituteby attribute values.
- Fixes some problems switching on debug mode.
- Fixes some src path problems in example html files.
